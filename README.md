Deep-Learning-Assisted Spectroscopic Single-Molecule Localization Microscopy based on Spectrum-to-Spectrum Denoising, Nanoscale, 16(11), 5729-5736.

## Usage
preparing the env.
```
pip install -r requirements.txt
```
traing.
```
python train.py

```
python test.py
